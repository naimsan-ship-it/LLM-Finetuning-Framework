# PEFT Fine-Tuning Project 🚀

Welcome to the PEFT (Pretraining-Evaluation Fine-Tuning) project repository! This project focuses on efficiently fine-tuning large language models using LoRA and Hugging Face's transformers library.

![](https://huggingface.co/datasets/trl-internal-testing/example-images/resolve/main/images/trl_overview.png)

# 🔬 LLM Fine-Tuning Framework

> **Maintained by [Naim San](https://github.com/naimsan-ship-it)**  
> Production-grade LLM fine-tuning with LoRA/QLoRA on Mistral-7B.  
> Achieved 60% inference cost reduction vs proprietary LLM APIs.  
> Deployed on AWS SageMaker with MLflow experiment tracking.

## 🛠️ What's Inside

- ✅ Mistral-7B, LLaMA2, Falcon fine-tuning notebooks
- ✅ LoRA & QLoRA parameter-efficient training
- ✅ RLHF training pipelines
- ✅ RAG with LangChain integration
- ✅ MLflow experiment tracking
- ✅ AWS SageMaker deployment configs
